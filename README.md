# Python Backend Foundations

Repositorio de estudio enfocado en fundamentos de Python con orientación a desarrollo backend.

## Objetivo

Construir una base sólida en:

- Tipos de datos
- Control de flujo
- Estructuras de datos
- Funciones
- Manejo de excepciones

Este repositorio forma parte de mi formación como desarrollador backend.

## Estructura

01_basics → Tipos de datos y variables  
02_control_flow → Condicionales y bucles  
03_data_structures → Listas, tuplas y diccionarios  
04_functions → Funciones y manejo de errores
